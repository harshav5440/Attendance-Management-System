# Attendance Management System

A simple Attendance Management System built using Java and MySQL.

## Tech Stack
- Java
- MySQL
- JDBC

## Features
- Daily attendance tracking
- Attendance percentage calculation using SQL
- CSV report generation
- Clean database design

## Database Setup
Run the SQL script from:
database/attendance_db.sql

## How to Run
1. Import project into IDE (IntelliJ / Eclipse)
2. Update MySQL username & password in AttendanceApp.java
3. Run AttendanceApp.java

## Sample Output

2 | Sita | 0%
3 | Aman | 100%
